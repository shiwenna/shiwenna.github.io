npm趋势对比图。能翻墙的小伙伴可以看一下Google Trends。综合各项对比都能看出ElementUI不论从周边生态、社区影响力、下载量等方面最有优势。

结论，vue下PC端我倾向的先后顺序是， ElementUI > iView > ant-design-vue > Vuesify。

npm趋势对比图。移动端的趋势并没有某一组件库特别突出，饿了么的Mint UI因已不维护所以并未列在其中。vux虽社区影响力最大、组件最多，但其文档、demo以及API的友好程度稍弱，并且目前属于个人维护，因此选择时应考虑风险。综合生态、组件丰富度vant会稍胜一筹。

结论，Mobile端我倾向的先后顺序是， vant > vux > cube-ui。

基于react的UI组件库-pc、Mobile端

适应于ReactJS的UI组件库有很多，例如：React Material-UI、React-Bootstrap、Semantic UI React、 Ant Design。这里只推荐 Ant Design，因其非常符合国人的审美需求，组件化质量非常高，开箱即用。生态较完善，具有Ant Design Pro、Ant Design Mobile等。
