## 新建
```
hexo new "my first blog"
```

## 生成静态文件
```
hexo generate//简写hexo g 
hexo g -d//文件生成后立即部署网站
hexo g -w//监视文件变动
```

## 发表草稿
```
hexo publish "my first blog"
```

## 启动本地服务器
```
hexo server [-p 4001] //可以修改端口
```

## 部署网站
```
hexo deploy
hexo d//简写
hexo d -g//部署之前先生成静态文件
```
## 清除缓存
```
hexo clean
```
